## Structure

Each example is a self-contained folder:

```
examples/
├── 01-primitives/
│   ├── README.md       # What this demonstrates
│   ├── index.ts        # Example code
│   └── expected.png    # Expected visual output (if applicable)
├── 02-booleans/
└── ...
```

## Running Examples

```bash
cd examples/01-primitives
npx ts-node index.ts
```

Or run all examples:

```bash
npm run examples
```

## Adding Examples

1. Create numbered folder: `NN-descriptive-name/`
2. Add `README.md` explaining what it demonstrates
3. Add example code
4. Update this index
